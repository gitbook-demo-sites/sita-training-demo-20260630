# SITA Product Hub

A fast-access knowledge layer for product training, solution design, sales enablement and product updates.

The discovery call pain is not lack of content. Content is spread across PDFs, PPTs, SharePoint, technical writer docs and Oracle LMS, so readers cannot tell which source is current or where to find a single fact.

{% hint style="info" %}
GitBook is positioned here as the layer before e-learning. Oracle LMS remains the home for deep SCORM packages; GitBook handles quick-access knowledge for employees, partners and contractors.
{% endhint %}

## Demo paths

1. Find a product and locate a quick fact.
2. Show categories inside a product: overview, sales, solution design, PM, delivery and support.
3. Ask the AI assistant a product question.
4. Open product updates to show RSS-style changelog content.
5. Show review and feedback flow for product stakeholders.
